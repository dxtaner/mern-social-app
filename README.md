
📱 Full-Stack MERN Social Media Platform
========================================

A comprehensive social media application built with the **MERN Stack** (MongoDB, Express, React, Node.js). This project features a modern architecture for social networking, content sharing, and real-time interaction.

MongoDB Express.js React.js Node.js JWT Auth Redux/Context

🚀 Core Features
----------------

*   **User Authentication:** Secure Sign Up/Login using JWT and bcrypt password hashing.
*   **Feed & Posts:** Create, edit, and delete text/image posts with real-time updates.
*   **Engagement:** Like, unlike, and comment system for interactive posts.
*   **User Profiles:** Customizable profiles with bio, profile pictures, and follow/unfollow logic.
*   **Cloud Media:** Image uploads managed via Cloudinary or Multer.
*   **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile devices.

⚙️ Installation & Setup
-----------------------

### 1\. Prerequisites

Install Node.js and have a MongoDB database (Atlas or Local) ready.

### 2\. Environment Variables

Create a `.env` file in the backend directory:

PORT=880
MONGO\_URI=your\_mongodb\_connection\_string
JWT\_SECRET=your\_super\_secret\_key
    

### 3\. Running the App

Install dependencies in both directories and start the servers:

\# Backend
cd backend && npm install && npm start

# Frontend
cd frontend && npm install && npm start
